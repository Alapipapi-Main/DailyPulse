# **App Name**: DailyPulse

## Core Features:

- News Fetching: Fetch news from external APIs based on supported categories via a secure Firebase Cloud Function proxy.
- Interest Selection: Allow users to select their interests (Technology, AI, Sports, Business, Gaming, Education, Health).
- Personalized Feed: Load and display a personalized news feed based on user-selected interests stored in Firestore.
- AI Summarization: Generate concise 1â€“2 sentence summaries of news articles using puter.js.
- Article Display: Display news articles in a card-based layout with headline, AI summary, source, publish time, and save button.
- Read Later Functionality: Allow users to save articles for later reading and store them in Firestore.

## Style Guidelines:

- Primary color: Desaturated blue (#6699CC) for a calm and modern feel.
- Background color: Light gray (#F0F0F0) to ensure a distraction-free environment.
- Accent color: Muted orange (#D98850) for interactive elements and highlights, ensuring they stand out without overwhelming the user.
- Headline font: 'Belleza' sans-serif, which will impart some personality aligned with the fashion, art, and design space.
- Body font: 'Alegreya' serif for body text, paired with 'Belleza' for the headers.
- Use simple, outline-style icons for categories and actions, ensuring clarity and a clean aesthetic.
- Employ a card-based layout with rounded corners and soft shadows for a modern and visually appealing presentation.
- Implement subtle loading animations and transitions for a smooth and responsive user experience.
